# I.基本觀念
## 資料階層
```
Bit（位元）→Byte（字元）→Field（資料欄）→Record（資料錄）→Table（資料表）→Data Base（資料庫）
```
## 資料表
由許多資料紀錄組成
## 資料庫
```
由許多資料記錄組成
```
## 資料與資訊
```
資料是未經處理的項目
資訊是經過處理的資料
```
## 檔案處理系統(file processing system) vs. 資料庫(database approach)
### 檔案處理系統
```
優點:
     簡單
     對記憶體負擔小
     被駭入時受到的傷害小
缺點
     資料重複性高
     更新複雜且易出錯
     資料不共享
```
